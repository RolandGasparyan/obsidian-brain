PROJECT GOAL
--------------------------------------------------------------------

====================================================================
STEP 3 — INSTALL DEPENDENCIES
--------------------------------------------------------------------
Run in Replit Shell:

pip install requests aiohttp asyncio python-dotenv

(If you use official SDKs, install them additionally.)

--------------------------------------------------------------------


====================================================================
STEP 4 — LOAD MASTER PROMPT (LOGIC)
--------------------------------------------------------------------
ALL AI MODELS must load the SAME prompt file:

ai_world_short_trading_championship.txt

Each model reads the prompt like this (concept):

MASTER_PROMPT = open(
    "ai_world_short_trading_championship.txt"
).read()

Each model receives:
• The master prompt
• Current market data
• No information about other models

--------------------------------------------------------------------


====================================================================
STEP 5 — DRY RUN (NO REAL TRADES)
--------------------------------------------------------------------
Test reasoning ONLY (recommended first):

python run_championship.py --mode dry

Expected output example:

[GPT-5] Decision: NO TRADE | Confidence: 74%
[Claude] Decision: SHORT | Confidence: 81%
[DeepSeek] Decision: NO TRADE | Confidence: 68%
[Llama] Decision: SHORT | Confidence: 77%
...

If this works → continue.

--------------------------------------------------------------------


====================================================================
STEP 6 — START LIVE CHAMPIONSHIP (REAL GATE.IO)
--------------------------------------------------------------------
When dry run is stable:

python run_championship.py --mode live

What happens:
• 8 AI models think independently
• Orchestrator collects decisions
• Gate executor opens SHORT trades
• leaderboard.json updates in real time

--------------------------------------------------------------------


====================================================================
STEP 7 — RUN 24/7 (BACKGROUND MODE)
--------------------------------------------------------------------
To keep championship running non-stop:

nohup python run_championship.py --mode live &

Check status:

ps aux | grep run_championship

--------------------------------------------------------------------


====================================================================
STEP 8 — STOP EVERYTHING (EMERGENCY)
--------------------------------------------------------------------
To stop all championship processes:

pkill -f run_championship
pkill -f python
